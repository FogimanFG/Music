# YouTube Music Player

Веб‑приложение для прослушивания музыки через YouTube. Позволяет искать треки, создавать плейлисты и управлять воспроизведением.

## Возможности

- Поиск музыкальных видео на YouTube по названию или исполнителю.
- Создание и сохранение пользовательских плейлистов.
- Управление воспроизведением (play/pause, next, previous, громкость).
- Прогресс‑бар с возможностью перемотки.
- Режим повтора и перемешивания треков.

## Технологии

- **HTML5** — структура приложения.
- **CSS3** — стильное оформление интерфейса.
- **JavaScript** — логика работы и взаимодействие с YouTube API.
- **YouTube IFrame Player API** — воспроизведение аудио.
- **YouTube Data API v3** — поиск и получение информации о видео.

## Использование

1. Откройте `index.html` в браузере.
2. Используйте строку поиска для нахождения музыки.
3. Добавляйте треки в плейлист.
4. Управляйте воспроизведением с помощью кнопок управления.
